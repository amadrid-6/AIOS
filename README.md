# AIOS

> An AI-native Operating System built from scratch.

## Vision

AIOS es un sistema operativo desarrollado desde cero para comprender en profundidad cómo funciona un sistema operativo moderno y explorar cómo integrar inteligencia artificial como parte de su arquitectura.

## Objectives

- Comprender el proceso de arranque de un ordenador.
- Implementar un bootloader propio.
- Construir un kernel desde cero.
- Implementar gestión de memoria.
- Implementar interrupciones.
- Implementar multitarea.
- Diseñar una arquitectura preparada para IA nativa.
