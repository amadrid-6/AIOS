# Day 00 - Development Environment

## Goal

Preparar el entorno de desarrollo para AIOS.

## Completed

- Instalación de VirtualBox.
- Instalación de Ubuntu en máquina virtual.
- Configuración de Guest Additions.
- Configuración de portapapeles compartido.
- Configuración de carpeta compartida entre Windows y Ubuntu.
- Creación del workspace AIOS.
- Creación de la estructura inicial del repositorio.
- Inicialización de Git.
- Creación del archivo `.gitignore`.

## Notes

El desarrollo se realizará principalmente en Ubuntu, usando Windows como sistema anfitrión y Obsidian como base de conocimiento.
Se ha definido una propuesta de trabajo de la siguiente manera:

- Teoría: explicación (20 - 30 min)
- Implementación: escribimos el código desde cero. El lema es no avanzar sin entender.
- Experimentación: modificamos código y vemos la respuesta.
